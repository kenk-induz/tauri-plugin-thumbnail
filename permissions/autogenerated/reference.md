## Default Permission

Default permissions for the thumbnail plugin

#### This default permission set includes the following:

- `allow-get-thumbnail`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`thumbnail:allow-get-thumbnail`

</td>
<td>

Enables the get_thumbnail command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`thumbnail:deny-get-thumbnail`

</td>
<td>

Denies the get_thumbnail command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`thumbnail:allow-ping`

</td>

</td>
</tr>

<tr>
<td>

`thumbnail:deny-ping`

</td>
<td>

</td>
</tr>
